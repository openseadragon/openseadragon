OpenSeadragon blendTime / alwaysBlend / layered opacity comparison

This page compares classic single-image blending with a stacked setup where multiple TiledImage instances sit on top of each other using per-image opacity. The layered viewers use identical bounds but different generated tile art, so you can see how semi-transparent layers behave while tiles blend in.

Suggested checks

  • Compare the single-image viewers against the layered viewers while rapidly wheel-zooming.
  • In the layered viewers, watch how underlying layers remain visible because each TiledImage uses a reduced opacity instead of alpha baked into the tile pixels.
  • Compare equal blendTime values with alwaysBlend off vs on in the layered presets.
  • tile-drawn stats are most useful with the canvas drawer; the WebGL drawer does not raise that event.
