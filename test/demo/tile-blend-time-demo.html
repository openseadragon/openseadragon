OpenSeadragon blendTime / alwaysBlend comparison

This page is meant for visually testing how tile replacement feels while zooming and panning. The first four viewers are fixed presets; the fifth viewer is a sandbox you can rebuild with your own settings. Enable sync to keep all viewers on the same viewport while you interact with any one of them.

Suggested checks

  • Compare blendTime: 0 against a longer blend while rapidly wheel-zooming.
  • Compare equal blendTime values with alwaysBlend off vs on.
  • tile-drawn stats are most useful with the canvas drawer; the WebGL drawer does not raise that event.
  • If you open this outside the OpenSeadragon repo, swap the source to the public DZI button or another valid DZI/IIIF source.
